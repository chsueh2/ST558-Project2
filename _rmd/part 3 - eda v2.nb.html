Response Variable

Next, lets learn more about our response variable: shares with a 5-number summary. With a histogram, we can visually see its distribution and determine if it is symmetric, skewed left, or skewed right.

# 5-number summary on sahres
summary(df_train$shares)
   Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
     28    1100    1700    3634    3200  208300 
# histogram on shares
df_train %>% 
  ggplot(aes(x = shares)) + 
  geom_histogram(binwidth = 1000) +
  coord_cartesian(xlim = c(0, 100000)) +
  labs(
    title = paste0("Histogram: Distribution of Shares for ", params$channel),
    x="Shares",
    y="Frequency")

Categorical Predictors

Despite the channel variable, there are two categorical variables weekday, is_weekend. A one-way contigency table shows how many articles were published on each weekday.

  summary()

The following side-by-side box plots help us visualize the distributions of shares on each weekday.

df_train %>% 
  ggplot(aes(weekday, shares)) +
  geom_boxplot() +
  scale_y_log10() +
  geom_jitter(width = 0.05) +
  labs(
    title = paste0("Boxplots: Distribution of Shares in ", params$channel, " for Each Weekday"),
    x = "Weekday")

Numeric Predictors

We have 45 numeric variables. It will be easier to inspect their distributions visually with histograms. We can look at each variable and see if it has a symmetric, skewed left, or skewed right distribution.

df_train %>% 
  select(-c(shares, channel, weekday, is_weekend)) %>% 
  hist.data.frame(mtitl = paste0("Histograms for all Numeric Variables in ", params$channel))
